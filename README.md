# About

Create Shafts is a simple mod that adds both Granite and Diorite alloys, enabling you to create shafts of that variety!

# Content
![Creative Tab](https://i.imgur.com/O4tzyOs.png)

- Diorite Alloy
- Granite Alloy
- Diorite Shaft
- Granite Shaft
